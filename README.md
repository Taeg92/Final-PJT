### ⚙️INSTALL

👨‍💻Back-End

```bash
$ cd backend
$ python -m venv venv
$ source venv/Scripts/activate #윈도우 환경
$ source venv/bin/activate #리눅스 환경
$ pip install -r requirements.txt
```

​     

👩‍💻Front-End

```bash
$ cd frontend
$ npm install
```
